NapkinGIS. User-friendly mapping.

NapkinGIS is a web-based Geographical Information System ( GIS ) software built to get the job done faster, easier and cheaper. Learn more about NapkinGIS in the details below.

The power of GIS. At your fingertips.

We bring you the power of GIS, fully integrated with all your business systems. Through the NapkinGIS software, we offer a variety of fully interconnected GIS software packages. This makes NapkinGIS unrivaled in both feature-set and extendability. No workload is out of NapkinGIS' reach.

GIS. What is it?

GIS is a common term used for all geospacial data-processing software. GIS is used in everything from space-travel to logistics. Watch this Ted Talk explaining GIS.

All the features you need.




Cross-platform

Windows, Mac, Linux, Mobile - NapkinGIS is web-based and available on all platforms.


Compatibility

NapkinGIS is compatible with all existing GIS software, right out of the box. No need for tedious migration processes.


OSS software

NapkinGIS is built exclusively with OSS software, making it a product of the best solutions the community has to offer.


Open standards

Web Map Services, Tile Map Services, Web Feature Services. NapkinGIS proudly supports all OGC-compliant standards.
